# Cave Dweller ReEvolved - Issues
### Official mod page: https://modrinth.com/mod/cave-dweller-reevolved

## Overview
This is the issue repository. All issues or suggestions should be made here. When opening an issue, please consider keeping your text well-formatted and structured. Describe the issue explicitly and in detail. And always specify if it's a problem or suggestion in the title (as well as with a brief description).

## Issues
When reporting an issue/bug, you should follow this structure:
1. Self-descriptive title.
2. Give a brief description of the issue/bug.
3. Provide the steps to reproduce it.
4. In case you don't know how to reproduce the bug, detail the situation in which it happened.
5. Provide the logs (only necessary if it is a crash).
6. Try providing the modlist (optional).

In case you don't follow these rules, the issue **will be ignored and immediately closed**.
## Suggestions
If you want to give a sugestion, consider following these rules:
1. Self-descriptive title.
2. You should give a brief description of your suggestion or idea.
3. Suggestions should simply be improvements or features, which fit the theme of the mod.

In case any of this rules aren't followed, the suggestion **will be ignored and immediately closed**.
